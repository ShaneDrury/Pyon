Performs the fit. A specific implementation will use a certain method of minimizing the chi-squared of fit_func against the data.

Parameters:
  • data – Iterable set of independent measurements that the fit function models.
  • errors – Iterable set of errors of each measurement.
  • initial_value (iterable) – Initial guesses of the best fit of the parameters in fit_func.
  • fit_range (iterable or list of iterables) – Iterable set of numbers that determine the data points used in the fit. Can be a list of iterables if the fit function requires more than one fit range e.g. simultaneous fit.
  • covariant (bool) – Set the fit as covariant.
  • correlated (bool) – Set the fit as correlated.

pyon.lib.plotting module¶

The point of this module is to be able to easily plot lattice QCD things - most often this will be correlators. We want to have the same syntax for common use plotting regardless of what program is used to plot it. So, the classes here implement a common API that forwards stuff to e.g. Pyplot and Gnuplot

class pyon.lib.plotting.Plotter(*args, **kwargs)[source]¶

Bases: builtins.object

A Plotter plots hadrons like mesons, baryons. e.g.

>>> from pyon.lib.meson import PseudoscalarMeson
>>> from pyon.lib.fitfunction import pp_flat
>>> data = [
...    [pp_flat(t, 1.0, 1.0, 64) for t in range(64)]
...    for _ in range(200) ]
>>> had = PseudoscalarMeson(data, masses=(0.05, 0.05))
>>> plot_hadron(had)

class pyon.lib.register.Register(register_to, name)[source]¶

Bases: builtins.object

Decorator to register the class cls to the dict register_to_cls with key name. Maintains a list of functions that can be used.

>>> from pyon.lib.fitting import registered_fitters
>>> @Register(registered_fitters, 'scipy')
... class ScipyFitter:
...     pass
>>> registered_fitters['scipy']
<class 'pyon.lib.fitting.ScipyFitter'>
